## 참여 인원
  - 한승록 외 4명

## 1. 프로젝트 개요
    1) Back-end
    2) Front-end
    3) Spring 라이브러리
    4) DataBase 테이블
    
## 2. 사용된 주요 기술
    1) 구현 기능
      ● DB구축
        - Crawling 기술 사용
      ● 게시판
        - select를 통한 DB불러오기
          -> 조건 설정을 하면 해당 조건에 대한 DB만 불러옴
      ● 페이징
        - 한 페이지당 10개씩 10페이지씩 불러옴
      ● 검색
        - 조건 설정을 통한 검색
        - Ajax를 통한 페이지 전환 없이 검색 가능
      ● 즐겨찾기
        - Ajax를 통해 페이지 전환 없이 즐겨찾기 가능
          -> 해당 버튼을 클릭하게 되면 비동기 통신으로 DB에 기록되고 삭제
